github issueに取り組むときの手順
1. ghコマンドで`https://github.com/Tetsu-is/social-media-scaling`のissueを取得する
2. 取得したissueをもとにユーザにどれに取り組んで欲しいのか選択肢を提示して選択してもらう
3. issueの詳細を読み込む
4. issueの解決のための設計・計画を作成してユーザーに提示する
5. 承認がもらえたら作業を開始する
6. 作業は新しいブランチを作成して行う
6. 実装が終わったら編集をadd,commit,pushしてprを作成する
